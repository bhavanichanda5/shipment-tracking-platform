# Shipment Tracking & Delivery Visibility Platform

A full-stack web application developed as part of the Infosys Springboard Virtual Internship 7.0.

##  Tech Stack

### Frontend
- React.js

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA

### Database
- PostgreSQL

### Authentication
- JWT Authentication
- OAuth2 Login

##  Project Modules

1. User Authentication & Role-Based Access
2. User Management
3. Shipment Management
4. Shipment Tracking
5. Live Delivery Monitoring
6. ETA Prediction
7. Route Management
8. Proof of Delivery
9. Notification Management
10. Analytics Dashboard
11. Reports & Export
